# OUTPUTS

```c#
using zx#.system

print {
  "Hello World!";
}
```

```

```

# HOW TO RUN IT
```
ZX# run script.zx
```

# VARIABLES

```c#
using zx#.system
using zx#.variables

x: sharp
z: a

print {
  x, z
}
```

OUTPUT:
```
ZX# run variable.zx
```

```
sharp a
```
